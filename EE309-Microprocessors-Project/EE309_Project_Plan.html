// 02 — Signal Generation

Step 1 — DAC Wiring & Single-Tone Sine Wave

DAC0808 Pin Connections

Connect all 8 data pins of DAC0808 directly to Port 1 (P1) of the 8051. P1 is a full-push-pull 8-bit output with no alternate functions conflicting with data.

DAC0808 PinConnect toNote
D0–D7 (pins 5–12)P1.0–P1.7D7 = MSB → P1.7
Pin 14 (Vref−)GND via 5kΩ to −5VSets Iref ≈ 2 mA
Pin 15 (Vref+)+5V via 5kΩReference voltage
Pin 16 (Vcc)+5VDigital supply
Pin 13 (Vee)−5VNegative supply
Pin 4 (Iout)LM741 inverting inputOp-amp feedback Rf = 5kΩ to Vout
Pin 3 (GND)Analog GNDStar ground

Op-amp output voltage formula:

Vout = −Iref × Rfeedback × [D7/2 + D6/4 + ... + D0/256]

With Iref = 2mA, Rf = 5kΩ → Vout swings 0 to −5V (inverted). Add a unity-gain inverting stage or swap the reference polarity to get 0–5V.

Single-Tone Lookup Table (C)

sine_gen_single.c C — 8051 
#include <reg51.h>

/* 36-sample sine table — 0° to 350° in 10° steps
   Formula: val = 128 + 127*sin(theta)
   Midpoint 128 = 0V (DAC centred) */
code unsigned char sine36[36] = {
  128,150,172,192,210,226,239,248,254,255,
  254,248,239,226,210,192,172,150,128,106,
  84, 64, 46, 30, 17,  8,  2,  0,  2,  8,
  17, 30, 46, 64, 84,106
};

/* Timer 0 reload value controls output frequency
   f_out = F_clk / (12 × T_reload × N_samples)
   For 12MHz crystal, N=36, f_out=500Hz:
   T = 12e6/(12*500*36) = 55.6 → TH0 = 256-56 = 200 */
#define TH0_RELOAD 200
#define N_SAMPLES  36

volatile unsigned char idx = 0;

void timer0_isr(void) interrupt 1 {
  TH0 = TH0_RELOAD;         /* reload */
  P1  = sine36[idx];        /* write sample to DAC */
  if (++idx >= N_SAMPLES) idx = 0;
}

void main(void) {
  TMOD = 0x02;             /* Timer 0, Mode 2 (8-bit auto-reload) */
  TH0  = TH0_RELOAD;
  TL0  = TH0_RELOAD;
  ET0  = 1; EA = 1; TR0 = 1;
  while(1);
}
📐 Frequency formula: f_out = F_clk ÷ (12 × (256 − TH0_RELOAD) × N_samples). For a 12 MHz crystal, 1 machine cycle = 1 µs. Adjust TH0_RELOAD to tune output frequency.
⚠️ Add an RC low-pass filter (R = 1kΩ, C = 10nF, cutoff ~16kHz) between the op-amp output and the BJT base to smooth DAC staircase artifacts before they reach the nonlinear circuit.
// 03 — Two-Tone Generation

Step 2 — Dual Phase Accumulator

Maintain two independent phase counters. On each timer interrupt, advance both, look up both sine values, average them, and output. This produces x(t) = A[cos(2πf₁t) + cos(2πf₂t)] digitally.

two_tone_gen.c C — 8051 
#include <reg51.h>

/* 64-sample sine table (0–255, midpoint=128) */
code unsigned char sine64[64] = {
  128,140,153,165,177,188,199,209,218,226,233,239,
  244,248,251,253,254,253,251,248,244,239,233,226,
  218,209,199,188,177,165,153,140,128,116,103, 91,
   79, 68, 57, 47, 38, 30, 23, 17, 12,  8,  5,  3,
    2,  3,  5,  8, 12, 17, 23, 30, 38, 47, 57, 68,
   79, 91,103,116
};

/* Choose step values for desired frequency ratio:
   f_tone = (step × Fs) / 64
   Fs = F_clk / (12 × (256-TH0_RELOAD))
   Example: Fs=8192Hz, step1=4 → f1=512Hz, step2=5 → f2=640Hz
   IM2 at |f2-f1|=128Hz, IM3 at 2f1-f2=384Hz */
#define STEP1  4
#define STEP2  5
#define TH0_R  137   /* 256 - 119 → ~8192Hz sample rate at 12MHz */

volatile unsigned char ph1 = 0, ph2 = 0;

void timer0_isr(void) interrupt 1 {
  unsigned int s;
  TH0 = TH0_R;
  ph1 = (ph1 + STEP1) & 0x3F;   /* mod 64 */
  ph2 = (ph2 + STEP2) & 0x3F;
  s   = (unsigned int)sine64[ph1]
      + (unsigned int)sine64[ph2];
  P1  = (unsigned char)(s >> 1); /* average → keeps in 0–255 */
}

void main(void) {
  TMOD = 0x02;
  TH0  = TH0_R; TL0 = TH0_R;
  ET0  = 1; EA = 1; TR0 = 1;
  while(1);
}
🎯 Choosing step values: Pick step1 and step2 as small integers with no common factor (coprime). The IM products land at integer multiples of (Fs/64) — all cleanly resolvable without spectral leakage.
// 04 — Non-Linear Element

Step 3 — BJT Common-Emitter Amplifier

The BJT collector current follows an exponential: Ic = Is · exp(Vbe/Vt). Expanding around the Q-point, the small-signal model includes 2nd and 3rd order terms that generate intermodulation products.

Circuit (2N2222, Vcc = 9V)

+9V | RC = 2.2kΩ | ------+----------> Vout (to ADC + scope) | [C] 2N2222 [B]----[10µF]-----< Vin (from DAC + LPF) [E] | RE = 470Ω (unbypassed — keep some gain compression visible) | GND Bias network: R1 = 47kΩ (from +9V to Base) R2 = 10kΩ (from Base to GND) Output cap: 10µF in series to Vout node

Bias Point Verification

ParameterCalculationValue
Vb9 × 10k / (47k+10k)≈ 1.58 V
VeVb − 0.6≈ 0.98 V
IcVe / RE = 0.98 / 470≈ 2.1 mA
Vc9 − Ic × RC = 9 − 4.62≈ 4.4 V ✓ (good midpoint)
Small-signal gain−RC / RE = −2200/470≈ −4.7
🔬 Drive the BJT base with 200–500 mVpp per tone to push into the nonlinear region. Too small → IM products buried in noise. Too large → clipping (hard saturation) rather than smooth polynomial nonlinearity.
⚡ Simpler alternative: A single silicon diode (1N4148) in series with a 1kΩ load to GND gives a strong α₂ nonlinearity (I–V exponential) and is much easier to breadboard. Good for demonstrating the f₂−f₁ IM2 product clearly.

ADC Read Routine (C)

adc_read.c C — 8051 
sbit WR   = P3^5;
sbit RD   = P3^6;
sbit INTR = P3^4;

unsigned char read_adc(void) {
  P0   = 0xFF;       /* set P0 as input */
  RD   = 1;
  WR   = 0;          /* WR low  ↓ */
  WR   = 1;          /* WR high ↑ → rising edge starts conversion */
  while (INTR == 1); /* wait for INTR to go low (done) */
  RD   = 0;          /* enable output */
  return P0;         /* read 8-bit result */
}
⏱️ Timing critical: Call read_adc() from a Timer 1 ISR at exactly your chosen sampling rate Fs. The time between ADC samples must be constant for coherent sampling to work correctly.
// 06 — Tone Amplitude Measurement

Step 5 — Coherent Sampling & Goertzel

Method A — Simple Coherent Multiplication (from project sheet)

Sample the signal at exactly 2× the tone frequency. First sample at t=0, then multiply alternate samples by +1 and −1, sum and average. This extracts the amplitude of that specific tone.

A_tone ≈ (2/N) × Σ [ x(n) × (+1,−1,+1,−1,...) ]

For the 60 Hz (or your chosen f₂−f₁) IM2 product: sample at 120 Hz. For 350 Hz IM3: sample at 700 Hz. This only works if no other tones are at integer multiples of the target frequency.

Method B — Goertzel Algorithm (recommended for 8051)

Computes DFT power at a single bin — far less computation than full FFT. Run it for each tone of interest.

goertzel.c C — 8051 (integer version) 
/* Integer Goertzel for 8051 — avoids float
   k = target bin = round(f_target × N / Fs)
   coeff_Q15 = 2*cos(2π*k/N) in Q1.15 fixed point */

#define N_SAMP 256

unsigned long goertzel_power(unsigned char *buf,
                                int coeff_Q15) {
  long s0 = 0, s1 = 0, s2;
  int  i;
  for (i = 0; i < N_SAMP; i++) {
    /* s0 = x[i] - 128 (remove DC) + coeff*s1 - s2 */
    s2 = s1; s1 = s0;
    s0 = (((long)coeff_Q15 * s1) >> 14) - s2
         + ((int)buf[i] - 128);
  }
  return (unsigned long)(s1*s1 + s0*s0
         - ((long)coeff_Q15 * s1 >> 14) * s0);
}

/* Usage example — measure power at f1=480Hz, Fs=8192Hz, N=256:
   k = round(480*256/8192) = 15
   coeff = 2*cos(2π*15/256) = 2*cos(0.368) ≈ 1.932
   coeff_Q15 = round(1.932 * 16384) = 31654 */

void measure_tones(void) {
  unsigned char samples[N_SAMP];
  unsigned long P_f1, P_f2, P_im2, P_im3;

  /* Collect N_SAMP samples from ADC at Fs */
  collect_samples(samples, N_SAMP);

  P_f1  = goertzel_power(samples, 31654); /* f1=480Hz, k=15 */
  P_f2  = goertzel_power(samples, 29197); /* f2=640Hz, k=20 */
  P_im2 = goertzel_power(samples, 32737); /* f2-f1=160Hz, k=5 */
  P_im3 = goertzel_power(samples, 30591); /* 2f1-f2=320Hz, k=10 */

  /* Compare ratios — IMD3 = P_im3 / P_f1 in dB */
}

Coherent Sampling Condition

To avoid spectral leakage, ensure your captured record contains an exact integer number of cycles of each tone:

f_in / Fs = M_cycles / N_samples (M should be odd or prime)

Since you control both DAC output frequency and ADC sampling rate from the same 8051 clock, you can guarantee this by design — set your frequencies to land exactly on FFT bins.
